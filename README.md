# wakka810

I build things close to the machine.

Mostly runtimes, emulators, compatibility layers, and ideas that are slightly unreasonable.

## selected work

[**3dmigoto-arknights-endfield**](https://github.com/wakka810/3dmigoto-arknights-endfield)  
A minimal 3Dmigoto port for Arknights: Endfield.

[**DotMatrixPy**](https://github.com/wakka810/DotMatrixPy)  
A Game Boy emulator written in Python.
